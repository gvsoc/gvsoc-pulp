GVSoC Pulp documentation
========================

This is the user documentation for the GVSoC models of the **PULP-platform**
targets — the open RISC-V systems (PULP-open, Snitch, Spatz, Occamy, MemPool,
Siracusa, rv64, ...) modelled in the ``gvsoc/pulp`` module.

It is aimed at people who run applications on these virtual platforms: it
explains how to run a binary, then documents each target with the
ready-to-run examples bundled with it.

For everything that is not target-specific (the simulator engine itself,
generic commands, debugging features, traces, the GUI, ...) refer to the
rest of this manual.
