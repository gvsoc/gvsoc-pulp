GVSoC Pulp developer documentation
==================================

This is the developer documentation for the GVSoC models of the
**PULP-platform** targets (the open RISC-V systems modelled in the
``gvsoc/pulp`` module).

It complements the generic GVSoC developer manual (programming model,
component model, engine architecture and the generic tutorials) with
guides that are specific to the PULP targets and their models.
